# Customer Churn Analysis Dashboard

## Project Overview
This project analyzes customer churn behavior using SQL and Power BI.

## Tools Used
- Power BI
- SQL
- Excel

## Dataset Features
- CustomerID
- Gender
- SeniorCitizen
- Partner
- Dependents
- TenureMonths
- InternetService
- Contract
- PaymentMethod
- MonthlyCharges
- TotalCharges
- Churn

## SQL Analysis Performed
1. Total Customers
2. Churn Distribution
3. Churn by Contract Type
4. Churn by Payment Method
5. Churn by Internet Service

## Dashboard KPIs
- Total Customers
- Total Monthly Charges
- Average Tenure

## Key Insights
- Month-to-month contracts have higher churn.
- Electronic check customers show significant churn.
- Longer contracts reduce churn risk.
